Authentication API With Google oauth

The Authentication API is built with Node.js, Express.js, and MongoDB to provide secure user authentication. It supports JWT-based login and signup, along with Google OAuth 2.0 integration using Passport.js.

Through this project, I gained strong experience in authentication flows, token handling, and API security best practices.

URl Shortener

URL Shortener is a full-featured SaaS application built with Node.js, Express, MongoDB, EJS, Tailwind CSS, and Stripe. It enables users to create, manage, and track custom short links via a user-friendly dashboard. The platform offers a subscription model, allowing users to choose between free and premium plans, with Stripe integration handling secure payments and subscription management.

Administrators have access to a comprehensive admin panel to monitor user activity, manage subscriptions, and oversee the overall system. The application ensures a seamless user experience with responsive design and intuitive navigation, making it suitable for businesses and developers seeking to manage and monetize URL shortening services.

This project demonstrates proficiency in building scalable web applications with integrated payment systems, user authentication, and administrative controls, showcasing the ability to develop production-ready solutions with modern technologies.
